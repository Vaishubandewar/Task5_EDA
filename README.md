# Task 5 - Titanic EDA

This project contains Exploratory Data Analysis (EDA) on the Titanic dataset.

## Files in this repository
- task5_eda.ipynb : Jupyter Notebook containing the EDA process  
- titanic.csv : Dataset used  
- task5_51.pdf : Task PDF  

## Steps performed:
1. Loaded dataset  
2. Checked shape, info, missing values  
3. Cleaned missing values  
4. Performed analysis with graphs:  
   - Age distribution  
   - Survived vs Not Survived  
   - Fare distribution  

## Tools Used:
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
